# go_sudoku

Sudoku related utilities written in go

## Getting Started

This is the start of some go command line program to solve sudoko
puzzles using recursion.

### Prerequisites

[Go programming language](https://golang.org)

```
go run solve.go < puzzle1.txt 
```
